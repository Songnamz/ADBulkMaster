  # AD Bulk Master

  **AD Bulk Master** is a lightweight, portable Windows GUI tool built for IT administrators
  to manage Active Directory users, licenses, cloud sync, and OneDrive provisioning —
  all from a single, easy-to-use interface. No installation required. Just run the `.exe`.

  ---

  ## Features

  ### 1. Create Single Account
  Create a new Active Directory user account with full attribute configuration
  — name, username, password, OU placement, and more — in just a few clicks.

  ### 2. Bulk Create (CSV)
  Import a CSV file to create hundreds of AD user accounts at once.
  Saves hours of repetitive manual work.

  ### 3. Bulk Set Properties
  Update properties (department, title, manager, etc.) across multiple existing
  AD accounts simultaneously using a CSV input.

  ### 4. Edit Account
  Search for and edit an individual AD user account — modify attributes,
  reset passwords, enable/disable accounts, and more.

  ### 5. Password Settings
  Configure and enforce password policies for selected users or groups,
  including password resets and expiry settings.

  ### 6. Cloud Sync (Azure AD / Entra ID)
  Connect to your AD Connect server by name or IP and trigger a
  **Delta Sync** to push new on-premises changes up to Microsoft 365 / Entra ID
  instantly — without opening the server manually.

  ### 7. OneDrive Provisioning
  Pre-provision OneDrive for multiple users before their first login.
  Enter email addresses (one per line), connect to your SharePoint Admin URL,
  and provision all users in one click using Microsoft's official
  `Microsoft.Online.SharePoint.PowerShell` module.

  ### 8. License Management
  View your software license status, hardware ID, and license key.
  Manage or deactivate your license from within the app.
